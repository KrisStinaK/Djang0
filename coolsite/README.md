# Простого блога на Django

Этот проект — простой блог, созданный с помощью Django. Он позволяет публиковать статьи, просматривать список статей и отдельные страницы статей.

## Особенности

- Просмотр списка статей
- Просмотр отдельной статьи
- Административная панель для добавления и редактирования статей

## Технологии

- Django 4.x
- Python 3.8+

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/KrisStinaK/Djang0.git
cd coolsite
```
2. Создайте виртуальное окружение и активируйте его:
```
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```
3. Установите зависимости:
```
pip install -r requirements.txt
```

4. Выполните миграции:

```
python manage.py migrate
```

## Запуск

Запустите сервер разработки:
```
python manage.py runserver
```
Перейдите по адресу http://127.0.0.1:8000/ в браузере.
## Использование

* Перейдите на главную страницу, чтобы просмотреть список статей.
* В административной панели (по адресу /admin) добавляйте и редактируйте статьи.
